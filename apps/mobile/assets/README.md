# Assets Directory

Place the following image assets in this directory:

- `icon.png` - App icon (1024x1024px)
- `adaptive-icon.png` - Android adaptive icon (1024x1024px)
- `splash.png` - Splash screen image (1284x2778px for iPhone 14 Pro Max)
- `favicon.png` - Web favicon (48x48px)
- `notification-icon.png` - Notification icon for Android (96x96px)

You can generate these assets using the Expo Asset Generator or create them manually.
