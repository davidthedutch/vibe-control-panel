# Escal App Specification

**App Name**: Escal (Escalatie Guide)
**Type**: Consumer/Business Mobile & Web App
**Control Panel**: Vibe Control Panel (beheer de Escal app)
**Domain**: escal.nl

---

## Wat is Escal?

Escal is een DJ/Festival Events Discovery App met live features, buddy systeem, en event scrapers.

**Platforms**:
- Flutter Mobile app (iOS + Android)
- Express.js API met WebSocket support
- Beheerd via Vibe Control Panel

**Symbiose met Vibe Control Panel**:
- Control Panel beheert Escal via API Key auth
- Feature Registry: features aan/uit zetten vanuit Control Panel
- Real-time monitoring: live users, scrapers, analytics
